Introduction

Thank you for choosing jcthemes and purchasing JC Agency, an easy to use Bootstrap theme for creating one page, smooth scrolling, dynamic websites.

We do not cover basic Bootstrap framework componenets/features in this documentation. For specific Bootstrap framework markup and Bootstrap JavaScript plugin documentation, please check the official Bootstrap documentation.

Instead, we've focused on:

  1. how to setup and use the theme
  2. how to use and configure the included JavaScript plugins
  3. how to customise the theme to your needs

Getting stuck?

Installation

JC Agency is supplied as a standalone bundle of HTML, CSS, LESS, JavaScript and webfont files. These have been coded to work on both a local or production server environment.

Opening index.html in your browser will render the theme. The PHP-driven contact form does not work unless you upload the theme to a server, or setup a local server.

The file structure has been organised to maintain seperation between HTML pages, CSS files, and JavaScript plugins. These are contained in separate folders within an assets folder. You can change this structure, though it is recommended to keep it â€” the theme won't know where it's assets are unless you tell it otherwise!

Adding your own HTML to our themes couldn't be easier. All our Bootstrap-based themes use the standard HTML markup of Bootstrap. If you want to add headings, paragraph text, buttons, tables, navigation, and so on, these all work as you would expect using the Bootstrap 3.2 framework.

The theme includes a selection of heading styles, colors, and general theme styles to quickly alter the appearance of your website.

JavaScript plugins

Several features of the theme have been implemented using JavaScript and jQuery.

The JavaScript initialisation is included in the agency.js file, inside assets/js.

Bootstrap plugins

The entire collection of Bootstrap plugins is available for your use. These have been bundled into the bootstrap.min.js file.

Smooth Scrolling

This is used for the navigation links, to allow for smooth scrolling to sections on the page. You can alter the easing and scrolling speed.


            $(function() {
                $('a.page-scroll').bind('click', function(event) {
                    var $anchor = $(this);
                    $('html, body').stop().animate({
                        scrollTop: $($anchor.attr('href')).offset().top
                    }, 1500, 'easeInOutExpo');
                    event.preventDefault();
                });
            });

Isotope

A third party plugin called Isotope is used to generate a Masonry grid, and flexible animated filtering for the Portfolio section of the theme. It is an extremely powerful and dynamic plugin, which is fully documented.

            
            // Initialise Isotope plugin for Portfolio Filtering

            $(window).load(function() {
                // init Isotope
                var $container = $('.isotope').isotope({
                    itemSelector: '.portfolio-item'
                });
                $('#filters').on('click', 'button', function() {
                    var filterValue = $(this).attr('data-filter');
                    $container.isotope({
                        filter: filterValue
                    });
                });
                // change is-checked class on buttons
                $('#filters').each(function(i, buttonGroup) {
                    var $buttonGroup = $(buttonGroup);
                    $buttonGroup.on('click', 'button', function() {
                        $buttonGroup.find('.active').removeClass('active');
                        $(this).addClass('active');
                    });
                });
            });

Retina Images

Retina images are supported in this theme via the retina.js plugin. First, make sure the retina.js script is called for within your document. The plugin checks your server to see if there are any high-res versions of your picture available, which are identified by having the @2xmodifier on the image.

For example, if you have an image on your webpage that looks like this <img src="path/to/image.jpg">then create another image called image@2x.jpgalong with the original image and on certain devices the high-res image will be served in place of the normal image.

For more details, visit the retina.js plugin documentation.

Icons

A full icon set is available through Font Awesome, a set of scalable vector icons that can be easily customised via CSS. More than 470 icons are available in total!

You can place Font Awesome icons just about anywhere using the CSS Prefix fa and the icon's name. Font Awesome is designed to be used with inline elements (we like the tag for brevity, but using a is more semantically correct).

fa-camera-retro
fa-camera-retro

If you change the font-size of the icon's container, the icon gets bigger. Same things goes for color, drop shadow, and anything else that gets inherited using CSS.

CSS Animation

Animate.css is included by default, giving you access to a range of CSS animations that you can apply to any element in the theme.

Dynamically add animations using jQuery with ease:

$('#yourElement').addClass('animated bounceOutLeft');

You can also detect when an animation ends:

$('#yourElement').one('webkitAnimationEnd mozAnimationEnd MSAnimationEnd oanimationend animationend', doSomething);

Changing animation duration

You can change the duration of your animations, add a delay or change the number of times that it plays:

#yourElement {
  -vendor-animation-duration: 3s;
  -vendor-animation-delay: 2s;
  -vendor-animation-iteration-count: infinite;
}

Animation on scroll

JC Agency expands the functionality of Animate.css with additional JavaScript which animates specific elements when they are first scrolled to. If you scroll through the JC Agency theme, you'll see what we mean!

JavaScript:
<!-- ViewPortChecker plugin (used for animated sections) -->
<script src="assets/js/jquery.viewportchecker.js"></script>


$('.service-item').removeClass("hidden").viewportChecker({
    classToAdd: 'visible animated zoomIn',offset: 100
});

CSS animations not supported in Internet Explorer 9

Bootstrap exclusively uses CSS3 for its animations, but Internet Explorer 9 doesn't support the necessary CSS properties. Thus, there are no slide transition animations when using Internet Explorer 9. We have intentionally decided not to include jQuery-based fallbacks for the transitions.

Usage:

First make sure that your body tag has a page-top id. It should look like this: <body id="page-top">. This will make sure that when a user clicks on the logo it will bring them back to the top of the page.

Each link should be within a <li class="page-scroll">. This will activate the smooth scrolling jQuery and will bring you to the target section.

Each link within the navigation should target a section ID. For example, if one of your menu items is <a href="#about">About</a>then there should be a corresponding <section id="about">somewhere within your HTML.

Setting the Logo Image:

There are two places you will need to set the logo image. The first <span class="logo"><img src="path/to/logo.png" alt=""></span>determines the logo image when the menu bar is expanded. The second <span class="logo-collapse"><img src="path/to/collapsed-logo.png" alt=""></span>determines the logo image when the menu bar is collapsed after scrolling. If you want the logos to be the same, you can delete the second line of code. If you want them to be different however, the option is there.

For the scrolling to work, make sure that your links have a parent class of .page-scrolllike in the example code. Also, make sure that your link targets match a section ID somewhere on your page.

Customizing the Logo

The logo in the theme

In order to customize the logo, we recommend creating a logo with a transparent background that has a height of 50px. The logo in the sample is white, but if your page background is lighter in color, your logo can be darker.

Also, you will want to create a high-res version of your logo, or the same logo but double the size, for mobile devices.

For example, the sample logo provided in the template is named logo.png but it also has a high-res version named logo@2x.png that is served on mobile devices. The retina.js plugin is what makes this happen. With your logo, simply use the same @2x condition for your high-res version and the plugin will do the rest!

Usage

Slides are contained within .item divs, which are contained within .carousel-inner wrapper.

The slider is initialised using a HTML5 data attribute, data-ride="carousel" on the main #hero-carousel div.

Inline styles on each .item div define the background images for each slide. Alternatively, you can include an img tag within each slide.

Please see the official Bootstrap Carousel documentation for more information.

PHP scripts do not run without a server environment

PHP does not run in your browser - it must run from a server environment. Loading the index.html page from your computer will work, but PHP won't process. PHP:

The following PHP script handles the posting of data and creation/sending of an email to a specified email address in the $tovariable.

<?php
// Check for empty fields
if(empty($_POST['name'])        ||
   empty($_POST['email'])       ||
   empty($_POST['phone'])       ||
   empty($_POST['message']) ||
   !filter_var($_POST['email'],FILTER_VALIDATE_EMAIL))
   {
    echo "No arguments Provided!";
    return false;
   }
    
$name = $_POST['name'];
$email_address = $_POST['email'];
$phone = $_POST['phone'];
$message = $_POST['message'];
    
// Create the email and send the message
$to = 'yourname@yourdomain.com'; // Add your email address inbetween the '' replacing yourname@yourdomain.com - This is where the form will send a message to.
$email_subject = "Website Contact Form:  $name";
$email_body = "You have received a new message from your website contact form.\n\n"."Here are the details:\n\nName: $name\n\nEmail: $email_address\n\nPhone: $phone\n\nMessage:\n$message";
$headers = "From: noreply@yourdomain.com\n"; // This is the email address the generated message will be from. We recommend using something like noreply@yourdomain.com.
$headers .= "Reply-To: $email_address"; 
mail($to,$email_subject,$email_body,$headers);
return true;            
?>

/*============================================
Contact Map

function loadGoogleMap() {

    var mapPoint = {
                'lat': 53.3776901,
                'lng': -2.8649802,
                'zoom' : 10,
                'infoText':'
94 Redcliffe Way\
                            
Woolton\
                            
L25 4NT
',
                'linkText':'View on Google Maps',
                'mapAddress':'94 Redcliffe Way, Woolton, L25 4TN',
                'icon': 'assets/img/map_pin.png'
            };

    if($('#gmap').length){
    
        var map;
        var mapstyles = [ { "stylers": [ { "saturation": -100 } ] } ];
        
        var infoWindow = new google.maps.InfoWindow;
        
        var pointLatLng = new google.maps.LatLng(mapPoint.lat, mapPoint.lng);

        var mapOptions = {
            zoom: mapPoint.zoom,
            center: pointLatLng,
            zoomControl : true,
            panControl : false,
            streetViewControl : false,
            mapTypeControl: false,
            overviewMapControl: false,
            scrollwheel: false,
            styles: mapstyles
        }
        
        map = new google.maps.Map(document.getElementById("gmap"), mapOptions);
        
        var marker = new google.maps.Marker({
            position: pointLatLng, 
            map: map, 
            title:mapPoint.linkText,
            icon: mapPoint.icon
        });
        
        var mapLink = 'https://www.google.com/maps/preview?ll='+mapPoint.lat+','+mapPoint.lng+'&z=14&q='+mapPoint.mapAddress;
        
        var html = '<div class="infowin">'
                + mapPoint.infoText
                + '<a href="'+mapLink+'" target="_blank">'+mapPoint.linkText+'</a>'
                + '</div>';

        google.maps.event.addListener(marker, 'mouseover', function() {
            infoWindow.setContent(html);
            infoWindow.open(map, marker);
        });

        google.maps.event.addListener(marker, 'click', function() {
            window.open(mapLink,'_blank');
        });
        
    }
}
